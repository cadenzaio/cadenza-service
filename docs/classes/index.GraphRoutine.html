    Class GraphRoutine

    Represents a routine in a graph structure with tasks and signal observation capabilities. Routines are named entrypoint for a sub-graph, describing the purpose for the subsequent flow. Since Task names are specific to the task it performs, it doesn't describe the overall flow. Routines, therefore are used to assign names to sub-flows that can be referenced using that name instead of the name of the task(s). Extends SignalEmitter to emit and handle signals related to the routine's lifecycle and tasks.
