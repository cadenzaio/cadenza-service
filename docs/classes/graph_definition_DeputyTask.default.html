    Class default

    Represents a task that delegates execution of a routine to a remote system or service. The DeputyTask serves as a proxy to perform and track the progress of a remote workflow. It extends the Task class with additional delegation capabilities.

    Emits various meta-signals for monitoring delegation progress and resolution.
